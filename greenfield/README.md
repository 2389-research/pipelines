# Greenfield

Greenfield project validation suite — discovery, synthesis, review, and validation stages for evaluating new project ideas before committing to build.

## Pipelines

| Pipeline | Description |
|----------|-------------|
| `greenfield.dip` | Orchestrator — runs the full greenfield validation flow end-to-end. |
| `greenfield_discovery.dip` | Discovery stage — explores the problem space, identifies constraints and opportunities. |
| `greenfield_synthesis.dip` | Synthesis stage — generates candidate architectures and approaches. |
| `greenfield_review.dip` | Review stage — multi-model evaluation of synthesized candidates. |
| `greenfield_validation.dip` | Validation stage — final feasibility and risk assessment. |

## Usage

```bash
tracker greenfield/greenfield.dip
```
